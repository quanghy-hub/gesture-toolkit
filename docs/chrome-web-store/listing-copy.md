## Name

Gesture Toolkit

## Short Description

Quick web gestures, clipboard history, and quick search tools for Chromium.

## Detailed Description

Gesture Toolkit helps you move through the web faster without adding a heavy toolbar or extra page clutter.

Main features:

- Mouse and touch gestures for opening links, closing tabs, edge swipe scrolling, and quick page navigation
- Clipboard history panel for reusing copied text inside editable fields
- Quick Search bubble for selected text and large images
- Host blacklist so you can disable the toolkit on sites where you do not want it

This Chrome Web Store build focuses on browsing workflow tools only.
