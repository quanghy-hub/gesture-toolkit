# Privacy Policy Draft

`Gesture Toolkit` stores settings locally in the browser so the extension can remember user preferences and clipboard history.

## Data the extension stores locally

- Gesture settings
- Clipboard history and pinned clipboard items
- Quick Search settings
- Excluded host list

## Data the extension does not send to the developer

- Browsing history
- Page content
- Clipboard history
- Selected text
- Image URLs
- Authentication data

## Network behavior in this build

- The extension opens user-requested search URLs in new tabs.
- The extension does not send user data to developer-owned servers.
- The popup and quick-search UI do not fetch third-party icon assets.

## User controls

- Users can disable the toolkit on the current host from the popup.
- Users can disable each feature group from the popup.
- Users can clear clipboard history from the popup.
