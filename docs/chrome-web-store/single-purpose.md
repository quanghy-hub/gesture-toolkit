# Single-Purpose Statement

`Gesture Toolkit` helps users interact with web pages faster through three connected workflows:

1. Trigger browser actions with mouse and touch gestures.
2. Reuse copied text through an inline clipboard history panel.
3. Launch quick searches from selected text or images.

All features support the same purpose: reducing the number of manual steps required to navigate, search, and reuse information while browsing.

Not included in this store build:

- Video floating tools
- Video screenshot tools
- YouTube subtitle translation
- Forum layout customization
- Inline translation
- OCR and external translation service integrations
